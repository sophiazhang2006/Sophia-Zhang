A simple but nice header image generator
for your Github profile

How to use it?

  1. Create a nice github header image.
  2. Create your GitHub profile README
    following this guide.
  3. Upload your header to your profile repo.
  4. Add this line to your README:

    ![Header](./your-header-image-name.png)
